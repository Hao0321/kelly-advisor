# 凱利數學速查

> 給 skill 在 Mode 1 / Mode 2 計算時引用。所有公式都假設你**誠實**輸入估計值；
> 公式不會修正 garbage input——它只會放大它。這就是為什麼有 R3 + R4。

---

## 1. 基本凱利公式

> ⚠️ **三條公式的 `f*` 意義不同，不可互換數字！**（最常見誤用）
> - 賭注型 `f*` = **你會賠掉的那筆**佔 bankroll 的比例（輸了就全沒）→ 幾乎都 ≤ 1。
> - 投資型 / 連續型 `f*` = **曝險 / 部署**佔 bankroll 的比例（可 > 1 = 槓桿）；輸時實際損失 = f × L。
> - 同一個賭局用兩式算，數字會不同但其實一致：投資型 f × L = 賭注型 f（兩者「真正會賠掉的」相同）。
> 看到 f* 先問：「這是『我會賠掉的比例』還是『我曝險的比例』？」

### 賭注型（最常見）
你下注本金的比例 `f`。贏的話淨賺 `b × 下注額`（賠率 b-to-1），輸的話賠掉下注額。
（這裡 f = **你會賠掉的比例**，輸了就全沒 → f 幾乎都 ≤ 1。）

```
f* = (b·p − q) / b  =  p − q/b
```

- `p` = 勝率（你的估計）
- `q` = 1 − p = 敗率
- `b` = 淨賠率（賺賠比；贏一次賺幾倍下注額）

**範例**：丟一個 60% 正面的硬幣，賭正面，賠率 1:1（b=1，p=0.6）
```
f* = 0.6 − 0.4/1 = 0.20  → full Kelly 說押 20% 本金
                          → 實際押 ½ × 20% = 10%（R4）
```

### 投資型（賺賠不對稱）
贏時賺 `+W`（比例），輸時賠 `−L`（比例）。

```
f* = p/L − q/W
```

**範例**：一個你會反覆做的操作，p=0.5，贏賺 +50%、輸賠 −20%
```
f* = 0.5/0.2 − 0.5/0.5 = 2.5 − 1 = 1.5
```
⚠️ `f* = 1.5 > 1` 代表 full Kelly 要你**用槓桿押 150%**。這正是凱利會喊得很激進、
而紀律全在「打折 + ruin check」的原因。實際下注應 ≤ ¼ f*，且先過 R2 破產檢查。

> **為什麼投資型 f 可以 > 1，賭注型通常 ≤ 1？** 因為投資型**單筆不會全損**（這例輸只賠 20%），
> 所以理論上要押到 5 倍部位才會把本金虧光；賭注型輸了就是輸掉整筆下注（全損），
> 所以 f 幾乎都 ≤ 1。f > 1 = 數學在叫你用槓桿——**而你幾乎永遠不該照做**（打折 + ruin check）。

### 連續報酬（近似）
資產有期望**超額**報酬 `μ`（已扣掉無風險利率 r）、報酬變異數 `σ²`：

```
f* ≈ μ / σ²      （嚴格寫法：f* ≈ (μ − r) / σ²，μ 為總報酬時要先扣 r）
```
高波動會快速吃掉最佳下注比例——這就是「波動本身是一種成本」的數學。
⚠️ 這是常態/小報酬近似，重尾資產（如加密、選擇權）會嚴重高估 f*，務必再大幅打折。

## 2. 什麼時候 f* ≤ 0？

```
f* ≤ 0  ⟺  你沒有 edge（期望值 ≤ 0）
```
公式給負數或零 = **別押**。不要把負的 f* 解讀成「做空」，除非你明確在建模做空。
沒有 edge 還押 = 把錢餵給變異數。

## 3. 為什麼永遠用分數凱利（Fractional Kelly）

下注 `c × f*`（0 < c < 1）。關鍵結果：

| 下注比例 | 長期成長率 | 波動 / 回撤 | 評語 |
|---|---|---|---|
| **½ Kelly**（c=0.5） | ~**75%** of 最大 | ~**50%** | 甜蜜點，預設值 |
| Full Kelly（c=1） | 100%（最大） | 極大 | 理論最優，實務上回撤兇殘 |
| 1.5× Kelly | ≈ **75%**（同 ½ Kelly！）| **3×** of ½ Kelly | 最划不來：同樣成長卻 3 倍波動 |
| 2× Kelly | **≈ 0** | 毀滅級 | 正 edge 也長期不賺 |
| > 2× Kelly | **負** | 破產 | 長期幾乎必歸零 |

> ⚠️ 上表為**連續 / 高斯近似**下的對照（成長率 ∝ 2c−c²、波動 ∝ c）。離散大賠率情形為近似，非精確常數。
> 注意 1.5× Kelly 跟 ½ Kelly **成長率一樣（拋物線對稱於 c=1）**，但波動是 3 倍——這就是「永遠寧可不足、不要過度」最有力的證明。

兩個必記事實：
- **Full Kelly 的回撤**：大致有 ~50% 機率，在某個時點回撤到剩一半本金。多數人心理扛不住。
- **過度下注是頭號殺手**：你只要**高估自己的 edge**，就等於在 full Kelly 之上加碼 → 滑向毀滅區。
  因為你**一定**會高估 edge，所以預設打折不是保守，是把現實內建進來。

> 一句話：half-Kelly 用一半的波動換到四分之三的成長。這是你能拿到最划算的交易之一。

## 4. 破產 / 遍歷性（Mode 3 的數學根據）

凱利最大化的是「**長期幾何成長率**」，它的正當性建立在三個假設上：
1. 同一個 edge 的下注會**重複很多次**
2. 你可以**連續調整**下注額
3. 你**不會中途出局**（一旦歸零，遊戲結束，沒有「長期」可言）

**一次性 / 不可逆決策同時違反這三條**：
- 換工作、all-in 創業 = 樣本數 1，沒有「重複很多次」
- 你只活一次（非遍歷 / non-ergodic）：100 個人各 all-in，10 個發財 90 個破產，
  「群體期望」很好看——但**你不是那 100 個人，你是 1 個只活一次的人**。
  你破產了，不能從平行宇宙裡其他 9 個自己借錢。

所以 Mode 3 的正確動作不是「算一個更小的 Kelly」，而是**換問題**：
不問「期望值多少」，問「**最壞情況我會不會出局？**」——只要有不可恢復的歸零可能，
不管機率多小、上檔多大，f → 0。先活下來，其他再說。

## 5. 著名實驗：已知 edge，人類照樣賭到破產

Victor Haghani & Richard Dewey 的實驗，論文：*"Rational Decision-Making under
Uncertainty: Observed Betting Patterns on a Biased Coin"* (2016)。
- 61 位有金融背景的受試者，給真錢起始 $25，30 分鐘自由下注，獎金上限約 $250
- 硬幣**事先明說正面機率 60%**（一個巨大且已知的 edge）
- 賠率對玩家有利，理性策略是每次押固定小比例（Kelly-ish，約 ~20% 上限以內）

結果（論文廣泛引用數字；ship 前建議對原文再核一次精確值）：
- 約 **28% 的人賭到歸零**（$0）
- 只有約 **21% 的人**拿到上限獎金
- 更驚人（原論文）：**67% 的人一度押反面**（押在對自己不利的那邊）、**近 1/3 曾把全部 bankroll 押在一翻**

> ✅ 已對 arXiv 原論文核對（arxiv.org/abs/1701.01427）：61 人、$25 起始、$250 上限、28% 歸零、
> 21% 拿上限、67% 押過反面——全部命中。引用標 Haghani & Dewey (2016)。
> 「近 1/3 全押一翻」正是 Mode 3 / all-in tie-breaker 的真實佐證。

**啟示**：給你一個已知的、巨大的 edge，多數聰明人**還是**會因為下注紀律差而輸光。
問題從來不是「有沒有 edge」，是「**有沒有紀律按正確比例下注**」。
這就是 kelly-advisor 存在的理由——不是幫你找 edge，是幫你**別把有 edge 的牌打到破產**。

## 6. 計算時的誠實檢查清單

skill 每次算完，自問：
- [ ] p 是有根據的估計，還是為了想押而硬湊的數字？（R3）
- [ ] 有沒有標「這建立在你猜的 p 上」？輸出有沒有 ≤ 2 位有效數字？（R3）
- [ ] 有沒有打折到 ½ 或更低？（R4）
- [ ] 本金是 surplus，不是救命錢？（R5）
- [ ] 跑過「p 高估 15%」的壓力測試？
- [ ] 最壞情況會不會出局？會 → 推翻數字，f→0（R2）
