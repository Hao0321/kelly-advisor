# Kelly Advisor — 凱利公式 survival-first 決策工具

> 一個 [Claude](https://claude.ai) skill。把凱利公式（Kelly Criterion）變成一個會逼你想清楚、
> 會幫你防破產、必要時會**拒絕回答**的決策夥伴。
>
> **「活下來，才能複利。」** · *Survive first. Compound later.*

## 先講一個實驗

2016 年有人找了 61 位**有金融背景**的人，給真錢，讓他們賭一個**事先明說正面機率 60%** 的硬幣。
edge 巨大、已知、對玩家有利。結果呢？

- 約 **28% 的人賭到歸零**
- 只有約 **21%** 拿到上限獎金
- 還有不少人一度去**押反面**

**給你一個已知的、巨大的 edge，多數聰明人還是會輸光。**
問題從來不是「有沒有 edge」，是「**有沒有紀律按正確比例下注**」。

凱利公式就是那個「正確比例」的數學。這個 skill，就是幫你**真的用上它**——而且是用對的方式用。

---

## 它跟一般「凱利計算機」差在哪

市面上的凱利計算機讓你填 p 和賠率，吐一個數字。問題是：

1. **人生決策的真實機率沒人知道** → 你只能瞎猜一個塞進去 → 公式吐一個**看起來很科學、其實是瞎猜**的數字。這叫 false precision，是危險的。
2. **凱利會喊得很激進** → 照 full Kelly 全押，回撤兇殘，一次高估 edge 就滑向破產。
3. **不是每個決策都該用凱利** → 換工作、all-in 創業這種一次性、不可逆的決定，硬套公式 = 拿錘子敲螺絲。

這個 skill 用 **survival-first** 設計把這三個坑堵起來。

---

## 三模式（先分類，再決定用不用凱利）

| 決策類型 | 例子 | 模式 |
|---|---|---|
| **可重複下注** | 投資部位、廣告加碼、撲克 | **Mode 1 · Size**：算 fractional Kelly 部位 |
| **組合 / 配置** | 有限時間 / 資金分給多個機會 | **Mode 2 · Allocate**：跨注分配、單注設上限 |
| **一次性 / 不可逆** | 換工作、all-in 創業、搬家 | **Mode 3 · Survive**：**拒給數字** → 防破產清單 + 路由質化圓桌 |

> Mode 3 是這工具最重要的部分。**一個工具的尊嚴，在於它會說「我不該回答這題」。**

---

## 核心紀律

- **下檔優先**：先確認最壞情況不會讓你「出局」，再談上檔。出局風險 → 下注趨近 0。
- **永遠打折**：預設 ½ Kelly 或更低（half-Kelly ≈ 75% 成長率、一半波動）。因為你**一定**會高估自己。
- **只賭 surplus**：救命錢的 Kelly fraction 永遠是 0。
- **誠實**：不報明牌、不推薦標的、不保證報酬。它幫你**輸得起、活更久**，不幫你賺更多。

## 怎麼用

裝好後，描述任何「可能賺可能賠」的決策，或直接喊：

- 「**用凱利公式**幫我看這個部位該開多大」
- 「這個機會**值得 all-in 嗎**？」（→ 它會判斷成 Mode 3，拒絕梭哈，改幫你防破產）
- 「我有 100 小時要分給 7 個專案，怎麼配？」
- 「這筆下注**會不會讓我破產**？」

## 實際跑起來長這樣

**例一 — 它幫你算（Mode 1）**

> **你**：我有個會反覆做的策略，勝率估 60%，賺賠 1:1，部位開多大？
>
> **kelly-advisor**：
> 【分類】Mode 1（可重複下注）
> 【破產檢查】本金是可承受歸零的投資金 ✅
> 【full Kelly】f\* = 0.6 − 0.4/1 = **0.20**
> 【建議下注】½ × 0.20 = **約 10%** ← 用這個
> 【壓力測試】若勝率其實只有 52% → f\* 掉到 0.04（部位差 5 倍）
> 【洞察】「你估 60% 就該押 10%，不是梭哈。而且你對 60% 有多確定？沒那麼確定 → 押更少。」

**例二 — 它拒絕幫你算（Mode 3，英雄功能）**

> **你**：我在猶豫要不要離職去創業，用凱利算一下值不值得。
>
> **kelly-advisor**：
> 【分類】Mode 3 → **拒絕用凱利**
> 【為什麼】這是樣本=1、不可逆、機率你根本不知道、一大半是非金錢價值的決定。
> 硬塞 p 進公式 = 用數學的權威感包裝瞎猜，害你過度自信。
> 【改問】離職後跑道多長？最壞情況你死得了嗎？有沒有不 all-in 的版本（兼職過渡 / 留退路）？
> 【路由】「值不值得跳」交給質化圓桌，不是一條公式。
> 【洞察】「凱利公式拒絕回答這題——這正是它幫你的方式。」

**一般凱利計算機在例二會吐一個數字。這個不會。差別就在這。**
